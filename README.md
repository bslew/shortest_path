# shortest_path - Calculate shortest path between start and finish nodes on a graph.

Status: Devel

# Purpose
Fun only.

# Download

```
git clone https://github.com/bslew/shortest_path.git	
```

# Dependences

- gcc that yields at least c++17.
- CMAKE (>=2.6)
- boost

# Compilation

```
cmake .
```


```
make
```


## Use Examples

```
cd build/src
./shortest_path -c ../../examples/input.par
```
